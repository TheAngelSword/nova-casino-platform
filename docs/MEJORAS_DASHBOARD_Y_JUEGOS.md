## UX recomendado

- Usar tarjetas con imagen arriba o miniatura izquierda.
- Mantener fondo oscuro.
- Usar acentos violeta/cian/dorado.
- Agregar tooltips para estados.
- Usar tablas compactas para QA y bugs.
- Usar tabs dentro de cada juego: Overview, Creativo, Arte, Audio, Matemática, Desarrollo, Testing, Homologación, Certificación, Demos, Documentos.
